核心工具生态
课程涉及的关键框架与工具
🔗 LLM 框架
  • LangChain（Python 编排框架）
  • LangGraph（Agent 图编排）
  • LangChain4j（Java 集成）
  • LiteLLM（统一 API 代理）
🗄️ 向量与检索
  • Qdrant（高性能向量数据库）
  • pgvector（PostgreSQL 向量扩展）
  • Elasticsearch（混合检索）
📊 可观测与评估
  • LangFuse（LLM 可观测平台）
  • RAGAS（RAG 评估框架）
  • DeepEval（端到端评估）
⚛️ 前端 SDK
  • Vercel AI SDK（Streaming UI）
  • React Server Components
  • Vue Composables
🤖 Agent 工具
  • OpenAI Function Calling
  • MCP — Model Context Protocol
  • CrewAI（Multi-Agent 框架）
🔒 安全与部署
  • Guardrails（输出审核）
  • OWASP LLM Top 10
  • Docker / K8s 部署方案
